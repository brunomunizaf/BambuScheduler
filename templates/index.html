Guia de Configuracao ▼

1. Developer Mode (obrigatorio)

Na tela da impressora: Settings > General > Developer Mode > ON

Sem isso, a impressora rejeita comandos MQTT de print (erro 84033543 "assinatura invalida").

2. LAN Only Mode (opcional)

Na tela: Settings > General > LAN Only Mode

Desconecta da cloud Bambu. Opcional, mas garante que a impressora aceita somente comandos locais.

3. Dados necessarios (.env)

  • PRINTER_IP — Settings > Network na tela da impressora
  • PRINTER_ACCESS_CODE — Settings > LAN (codigo de 8 digitos)
  • PRINTER_SERIAL — Settings > Device Info ou adesivo fisico

4. Arquivo .3mf fatiado

O arquivo deve ser fatiado no Bambu Studio antes do upload.

No Bambu Studio: File > Export > Export plate sliced file (.3mf)

Arquivos .3mf nao fatiados (sem gcode) serao rejeitados pelo upload.

5. AMS Mapping

Selecione o slot da AMS correspondente ao filamento desejado.

Slots 0-3 correspondem as posicoes fisicas da AMS Lite, da esquerda para direita.

Desmarque "Usar AMS" para usar o filamento do spool externo.

6. Agendamento

Selecione uma data/hora futura e clique "Agendar". O servidor fara upload e enviara o comando no horario marcado.

O servidor precisa ficar rodando ate o horario agendado (nao feche o terminal).
