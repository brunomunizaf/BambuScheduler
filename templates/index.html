Setup guide

1. LAN Only Mode

On the printer screen: Settings › LAN Only Mode › ON. Disconnects from the Bambu cloud so the printer accepts local commands only — BambuScheduler can't reach the printer without it.

2. Developer Mode

On the printer screen: Settings › LAN Only Mode › Developer Mode › ON. Without it, the printer rejects MQTT print commands (error 84033543, "invalid signature").

3. Connection details

Enter these in the app's setup screen (the menu bar app), not a file:

  • IP address — Settings › LAN Only Mode on the printer
  • Access Code — Settings › LAN Only Mode (8 digits)
  • Serial Number — Settings › Device Info or the sticker

4. Slice your file first

The .3mf must be sliced in Bambu Studio before upload: File › Export › Export plate sliced file (.3mf). Unsliced files are rejected.

5. AMS mapping

Pick the AMS slot holding the filament you want. Slots 0–3 map left to right on the AMS Lite. Turn off "Use AMS" to print from an external spool.

6. Scheduling

Choose a future time and hit Schedule. BambuScheduler uploads the file and starts the print at that moment — the app needs to stay running until then.

Keep this Mac plugged into power and don't close the lid while a print is scheduled. If the Mac sleeps, the scheduled print may not fire. The screen going dark is fine — BambuScheduler keeps the system awake even while the display sleeps.
